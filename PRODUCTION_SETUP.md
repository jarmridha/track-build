# Production Setup Guide

## 1. Setup Supabase
- Create project in Supabase
- Copy URL and anon key
- Paste into .env

## 2. Run locally
npm install
npm run dev

## 3. Deploy
- Use Vercel or Netlify
- Add environment variables

## 4. Database Tables
- projects
- daily_updates
- activity_logs
- profiles

## 5. Security
- Enable RLS in Supabase
- Apply role-based policies

## 6. Optional upgrades
- File storage (images/documents)
- Email alerts
- PDF reports
