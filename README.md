# Track Build

Professional construction project tracking web app for engineers, supervisors, and admins.

## Features

- Secure login and signup
- Role-based access for Admin, Engineer, and Supervisor
- Admin user management
- Project dashboard with total, running, completed, and delayed projects
- Project tracking with start date, end date, progress, status, and remarks
- Daily progress update system
- Activity log for project and user actions
- Report page for project summary review
- Mobile-friendly responsive interface for field/site use
- Clean modern UI

## Tech Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Router
- TanStack Query

## Run Locally

```bash
gh repo clone jarmridha/track-build
cd track-build
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Purpose

Track Build is designed to make construction project monitoring simple, fast, and practical. It helps manage project status, daily site progress, users, reports, and activity history from one responsive dashboard.
